# Data Processor

A comprehensive data processing application with plotting and analysis capabilities.

## Features

- **Data Processing**: Advanced CSV data processing and analysis
- **Plotting**: Interactive plotting with zoom, pan, and export capabilities
- **Signal Management**: Signal list management and filtering
- **Export Options**: Multiple export formats (CSV, Excel, MATLAB)
- **Performance Optimized**: Optimized for large datasets

## Files

- `Data_Processor_r0.py` - Main application (most recent version)
- `Data_Processor_FUNCTIONAL_BASELINE_2025-07-29.py` - Functional baseline version
- `Data_Processor_BACKUP_BEFORE_2025-01-27_CHANGES.py` - Backup version
- `launch_app.py` - Application launcher
- `launch_backup.py` - Backup launcher
- `launch_debug.py` - Debug launcher
- `create_plotting_fix.py` - Plotting fixes and improvements
- `plotting_debug_patch.py` - Plotting debug patches
- `run_data_processor.bat` - Windows batch launcher
- `run_data_processor.ps1` - PowerShell launcher

## Usage

1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python launch_app.py`
3. Or use the batch file: `run_data_processor.bat`
